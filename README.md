## 전화상담 버튼 클래스

전화상담 버튼에 특정 클래스 추가

- pc : goTelOrderCounsel

```html
<a href="url" class="goTelOrderCounsel">전화상담신청</a>
```

- m : insInPageName

```html
<a href="url" class="insInPageName">전화상담신청</a>
```
- [ ] 사이를 한 칸 띄워 체크박스 만들기
